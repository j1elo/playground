long d_seek ( int  fd,
long  offset,
int  whence 
)

Posiciona el puntero de un fichero remoto en el offeset especificado.

Parámetros:
fddescriptor del fichero remoto (realmente es el descriptor de un socket).
offsetoffset que aplicar a la posicion de referencia en el fichero.
whenceposicion de referencia:
  • SEEK_SET para el principio del fichero.
  • SEEK_CUR para la posicion actual.
  • SEEK_END para el final del fichero.
Devuelve:
posicion actual en el fichero (puede no ser la pedida, en caso de error), -1 si error.
